# C-Plus-Project-Generator
Simple command-line tool for creating c++ projects with full folder structure and makefile.

## Build Status
| Platform           | Supported |
| :----------------- | :-------- |
| `Linux (x86_64)`   | Yes       |
| `macOS (arm64)`    | No        |
| `Windows (x86_64)` | Yes       |
